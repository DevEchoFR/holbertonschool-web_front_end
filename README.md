# holbertonschool-web_front_end

Front-end curriculum project from Holberton School focused on building the Techium website progressively with semantic HTML and modern CSS.

## Overview

This repository is organized as a step-by-step learning path:

1. Build page structure and content with semantic HTML.
2. Add visual styling and layout with CSS.
3. Improve consistency with style guide files and reusable patterns.

The numbered files represent incremental tasks. Later files typically include everything from previous steps plus new requirements.

## Main Learning Goals

- Write valid, semantic HTML5 markup.
- Use accessibility-friendly structure and attributes.
- Build responsive layouts with modern CSS.
- Apply reusable design tokens and component patterns.
- Maintain clean, readable, and progressively improved files.

## Quick Start

### 1) Clone the repository

```bash
git clone https://github.com/DevEchoFR/holbertonschool-web_front_end.git
cd holbertonschool-web_front_end
```

### 2) Open a page locally

```bash
xdg-open html_advanced/index.html
```

### 3) Serve with a local server (recommended)

```bash
python3 -m http.server 8000
```

Then open:

```text
http://localhost:8000/html_advanced/index.html
```
