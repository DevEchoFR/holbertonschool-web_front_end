# Holberton School - Web Front-End Curriculum

A comprehensive front-end web development project from Holberton School, focusing on HTML and CSS fundamentals through the progressive development of **Techium**, a digital agency website.

## 📋 Project Overview

This repository contains a complete learning journey for front-end web development, starting from semantic HTML markup to advanced CSS styling. Students work incrementally through numbered files to build a fully-featured website with proper HTML5 structure, accessibility, and modern CSS techniques.

## 🎯 Learning Path

### **Phase 1: HTML Advanced**
Build the foundational HTML structure of the Techium website with semantic markup and best practices.

**Key Concepts:**
- HTML5 document structure and semantic elements
- Metadata configuration (charset, viewport, description)
- Favicon setup
- Accessibility attributes (alt text, ARIA labels, language attributes)
- W3C HTML compliance

**Files Structure:**
- Progressive numbered files (0-index.html through 39-index.html) showing incremental development
- Complete page files (index.html, about.html, contact.html, article.html, latest_news.html)
- Styleguide pages for design consistency

### **Phase 2: CSS Advanced**
Style the Techium website using advanced CSS techniques and design patterns.

**Key Concepts:**
- Advanced selectors and CSS specificity
- Layout techniques (Flexbox, Grid)
- Responsive design principles
- CSS animations and transitions
- Design system implementation
- Component-based styling

## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor (VS Code, Sublime Text, etc.)
- Basic understanding of HTML and CSS

### Viewing the Project

**Option 1: Open in Browser**
```bash
# Navigate to the html_advanced folder
cd html_advanced

# Open any HTML file directly in your browser
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

**Option 2: Use Live Server (VS Code)**
1. Install the "Live Server" extension in VS Code
2. Right-click on an HTML file
3. Select "Open with Live Server"

**Option 3: Local Web Server**
```bash
# Python 3.x
python -m http.server 8000

# Then visit: http://localhost:8000/html_advanced/index.html
```

## 📚 Project Requirements

### HTML Standard
- ✅ DOCTYPE declaration on all files
- ✅ Proper HTML5 semantic structure
- ✅ Language attribute (`lang="en"`)
- ✅ Text direction attribute (`dir="ltr"`)
- ✅ Meta tags: charset, viewport, description
- ✅ Favicon configuration
- ✅ Proper heading hierarchy (h1-h6)
- ✅ Descriptive alt text on all images
- ✅ All files end with newline character
- ✅ W3C compliant HTML

- ✅ Modern CSS techniques
- ✅ Semantic class naming
- ✅ Performance optimization
- ✅ Cross-browser compatibility
