# CSS Advanced - Styles

This directory contains the incremental stylesheet files for the **Advanced CSS / Techium** project.

Each file represents one step in the progression of the project tasks:

- `1-style.css` to `32-style.css`

The idea is simple:

- Start from the previous file
- Add only the new rules required by the next task
- Keep all earlier rules unless the task asks for a change

## What These Files Cover

Across the sequence, the styles progressively introduce:

- CSS custom properties (variables)
- Typography systems (families, sizes, weights, line-heights)
- Link states and pseudo-classes
- Layout utilities (container, rows, columns, clearfix)
- Theme handling (`[data-section-theme="dark"]`)
- Components (buttons, cards, testimonial, services, works)
- Header and navigation behavior
- Pseudo-elements and transition effects

## How To Use

- Link the stylesheet that matches the task you are working on.
- Example in HTML:

```html
<link rel="stylesheet" href="styles/32-style.css" />
```

## Notes

- File names and order matter for project validation.
- These files are intentionally cumulative and educational.
- Some tasks mention that W3C validation is optional.
